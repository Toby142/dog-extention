## Run Locally

Clone the project

```bash
  git clone https://github.com/Toby142/dog-extention.git
```

Go to the project directory

```bash
  cd dog-extention
```

Install dependencies

```bash
  npm install
```

Try it

```bash
    Open a debug tab with F5 (in vs code)
```
